Console Log Test Page

Click the buttons below to generate different types of console logs. These logs should be captured by the Browser Console Capture extension and sent to the Cursor IDE.

Connection Instructions

  1. Make sure the server is running: npm start
  2. Click on the Browser Console Capture extension icon
  3. Ensure the Server URL is set to: ws://localhost:3000
  4. Click "Connect" and verify the status changes to "Connected"
  5. Click the buttons above to generate console logs
  6. Check Cursor IDE to see if the logs are being displayed

Troubleshooting

1. Check browser console for errors (F12)
2. Verify server is running on port 3000
3. Make sure extension is properly loaded
4. Try reloading the extension from chrome://extensions/
5. Ensure the WebSocket URL includes clientType parameter
   (ws://localhost:3000?clientType=browser)
